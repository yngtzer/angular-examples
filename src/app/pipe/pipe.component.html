Definition
  • Angular built-in pipes More...
  • Pipes can be configure by passing a string as a parameter after colons
  • Pipes can be chained
  • Custom Pipe
    1. Create ___.pipe.ts file
    2. import Pipe, PipeTransform from angular core
    3. Include @Pipe and declare object name: ___ inside
    4. Implements PipeTransform
    5. The function first parameter is compulsory transform(value, ...)
    6. Return a value
  • Updating an array doesn't trigger pipe to re-run, hence it's called Pure Pipe
  • Impure pipe refer to re-run of pipe whenever input values changes. As a result, performance is greatly affected
  • Async data i.e. Observable & Promise should use 'async' pipe
