Retrieving Params From Url
  1. import ActivatedRoute & Params from '@angular/router'
  2. inject
  3. Method 1: Non-reusable, this.route.snapshot.params['PARAM_NAME']
  4. Method 2: this.route.params.subscribe( (params: Params) => ... params['PARAM_NAME'])
  5. Params will be of type string

Route Links

Using HTML anchor tag to navigate to other components

  1. routerLink="..."
  2. [routerLink]="[....]"
  3. routerLinkActive="active"
  4. [routerLinkActiveOptions]="JSON_OBJECT(exact: true)"
Queries & Fragment

Passing queries parameters and fragment by specifying below attribute in an anchor tag

  1. [queryParams]="JSON_OBJECT(key: value)"
  2. fragment="..."
Merging | Preserving Current Params

Current params can be forwarded to the next URL

  1. queryParamsHandling="preserve OR merge"

Out-Sourcing Routes

Each feature module can declare it's own routes and be exported

  1. Create routing module file, ___-routing.module.ts
  2. Declare routes, const _____Routes: Routes = [.....]
  3. Include meta-data, @NgModule(JSON_OBJECT)
  4. JSON_OBJECT => imports: RouterModule.(forRoot OR forChild)(ROUTE_NAME)
  5. exports: [RouterModule]
