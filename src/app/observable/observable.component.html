Observable

Definition

Provide support for passing messages between publishers and subscribers. It is not executed until a consumer subscribes to it and capable to deliver multiple values of any type. Setup & tear down logic are both handled by the observable.
Publisher
  • Create an observable instance that defines a subscriber function
  • Subscriber function execute when consumer calls the subscribe() method, and passing an observer.
  • Subscribe() call returns a subscription object that has an unsubscribe() mthod to be call to stop receiving notification.
Observer
  • An Javascript object that defines the handlers for the notifications to be received.
  • next - (Required) A handler for each delivered value
  • error - (Optional) A handler for an error notification
  • () - (Optional) A handler for the execution complete notification
Subject

Can act as an publisher & observer!
