CanDeactivate Guard

Restrict user from leaving based on some conditions.

    
      { path: '____', canActivate: [SOME_GUARDS...], component: ______ }
Implementation
  1. Create a file ___-can-deactivate-guard.service.ts
  2. Create and export CanComponentDeactivate interface with a method canDeactivate: () => Observable of boolean | Promise of boolean | boolean
  3. Create CanDeactivateGuard class
  4. Import CanActivate, ActivatedRouteSnapshot, RouterStateSnapshot & Router from @angular/router
  5. Import Observable from rxjs
  6. Import injectable from @angular/core
  7. Implement CanDeactivate interface of CanComponentDeactivate
  8. Inject Router
  9. CanDeactivate(component: CanComponentDeactivate, currentRoute: ActivatedRouteSnapshot, currentState: RouterStateSnapshot, nextState?: RouteStateSnapshot): Observable of boolean | Promise of boolean | boolean { .... return component.canDeactivate(); }
  10. Implement CanComponentDeactivate interface on the desired component
